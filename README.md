# 🛒 E-commerce Project

## 🚀 Features

* User authentication (Login/Register)
* Admin dashboard
* Add/Edit/Delete products
* Cart system
* Checkout with Razorpay

## 🛠️ Tech Stack

* Node.js
* Express.js
* MongoDB
* Handlebars (HBS)

## ⚙️ Setup

```bash
git clone https://github.com/swalihch/ecommerce-project-1.git
cd ecommerce-project
npm install
```

Create a `.env` file:

```
MONGO_URI=your_mongo_uri
```

Run the app:

```bash
node app.js
```
